Summary

ISSN 2764-0582

 

A aplicação de técnicas de aprendizado de máquina na Física, tais como as redes neurais, tem se popularizado nos últimos anos. Uma destas aplicações é a PhysicsInformed Neural Networks (PINN) que é capaz de obter soluções numéricas para equações diferenciais. Neste trabalho a PINN foi utilizada para calcular soluções das equações de Einstein. Soluções de diferentes buracos negros e de um buraco de minhoca foram obtidas com boa precisão atestando a eficiência da PINN.
